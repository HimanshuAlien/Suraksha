# 🚀 Overview

SURAKSHA is an **AI-first Cyber Defense & Misinformation Intelligence System** designed to simulate a **national-level digital security infrastructure**.

It doesn’t just detect threats —  
it **investigates, cross-verifies, explains, and interacts** like a real cyber analyst.

---

# 🔥 5-Layer Cyber Threat Engine

1. **AI/ML Semantic Detection**
   - Detects phishing language, urgency patterns, impersonation

2. **VirusTotal Intelligence**
   - Global malware & URL reputation verification

3. **OSINT & Domain Intelligence**
   - WHOIS, domain age, registrar trust

4. **Lexical Analyzer**
   - Detects fake domains (microsOft, paypaI)

5. **AI Verdict Engine (Gemini)**
   - Generates reasoning + safety checklist

# ⭐ THE STAR: AI MISINFORMATION TRACKER (misinfo.html)

## 🧠 Not Guessing — Mathematical Cross-Verification

The system validates every claim using **6 global intelligence layers**:

---

## 🔍 1. Data Ingestion Layer (OSINT Intelligence)

### 🟢 Google Fact Check API
- Queries verified fact-checkers (Reuters, Snopes, Politifact)
- Returns verdicts like: *False, Misleading*

### 🌐 GDELT Database
- Detects propaganda patterns
- Global sentiment + event anomaly tracking

### 📰 News API + MediaStack
- Cross-checks live news coverage
- If no credible source → credibility drops sharply

### 🔎 SerpAPI
- Simulates real-time Google investigation
- Finds niche/local misinformation signals

---

## 🧠 2. AI Reasoning Layer

### 🤖 Gemini AI Engine
- Consumes ALL API outputs (JSON)
- Acts as digital forensic analyst
- Produces:
  - Probability of fake (0–100)
  - Explanation report
  - Source-backed reasoning

---

## 💬 3. Interactive Chat Intelligence UI

### Step-by-Step Flow:

1. User enters claim (WhatsApp forward / news / tweet)  
2. System queries all 6 APIs sequentially  
3. UI shows animated **Credibility Score (0–100)**  
4. Displays:
   - Verdict (Safe / Suspicious / Fake)  
   - Source evidence  
5. 🔥 **Continuous AI Chat (Major USP)**  
   - Ask follow-ups like:
     - “Who started this?”
     - “Is there proof?”
   - AI remembers context + responds intelligently
